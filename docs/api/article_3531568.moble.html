韩亚航空214号班机空难（OZ214/AAR214）是发生于2013年7月6日的一起航空事故。該班機是由一架韩亚航空波音 777-200ER營運的定期航线，从仁川国际机场飛往旧金山国际机场，机上共载有291名乘客和16名机组人员。在旧金山国际机场，飞机着陆失败，造成2人死亡，180人受伤[4]。事故原因目前仍在調查中，不过尚未发现有恐怖袭击的证据。[5][6][7][8]這是波音777客機投入商業運營18年後首次發生致命空難[9]。

韩亚航空214号班机为波音777-200ER，配备普惠PW4090引擎[11]，注册号为HL7742[12]，2006年3月交付韩亚航空服役[12][13][14]。机长李真珉（Lee Jeong-min，音译）有超过1.2万小时的飞行经验，驾驶波音777飞机的经验为3220小时。副机长李强国（Lee Kang-guk，音译）拥有近1万小时的飞行经验，驾驶777的经验为43小时。[4]

从仁川国际机场起飞[15]。旧金山当地时间2013年7月6日11时26分（UTC时间2013年7月6日18时26分），飞机在旧金山国际机场28L跑道即将着陆时坠毁[5][6]，飞机撞到机场旁边的海堤[16][5][6]，发动机和机尾脱落，机尾残骸散落在跑道上，机身滑行从左侧衝出跑道，最后停在离海堤610米的地方。[10]幸存的乘客称飞机降落时太靠近水面，美國國家運輸安全委員會（NTSB）负责人称飞行员反应太慢，在最后时刻试图加速提升飞机高度，但为时已晚。[4]目击者称飞机着陆瞬间出现了火球，几分钟后发生爆炸，浓烟從机身冒出。機組人員打開緊急逃生滑梯疏散乘客[17][18]。虽然受到強烈撞击，但许多乘客还可以自行離開飛機[19]。媒体照片显示飞机的紧急逃生滑梯在事故中打開，机身前部烧出大洞，飞机尾部則脱离了机身。[20]

當時飞机正按标准程序降落，降落时天气晴朗。由於机场正進行維修，可以提示機師飛機下降斜率的电子系统被关闭，至于这是否造成坠毁的原因則正在调查中。[4]

这是韩亞航空成立25年来的第三次致命空难[21]，也是波音777客机的首次致命空难。该机是第三架在事故中机身损毁的波音777客机[22]。

乘客和机组人员[编辑]

乘客和机组人员國籍分布见下表。事故造成两名16岁中国籍女性遇难，180人受伤[4]，其中6人伤势严重。伤者目前在当地的9家医院救治[23]。多数重伤的乘客都坐在飞机尾部，包括六名脊椎受损、麻痹和头部受伤的危重伤者。尚不清楚死亡的两名死者的位置，她们的尸体都是在机外找到的[4]，其中一名死者懷疑因沒有扣上空全帶而被拋出機外[24]，另有怀疑一名死者是被後來的救援車輛輾過壓死，具体原因正在调查[25]。遇难者的姓名是王琳佳和叶梦圆，她们都是浙江江山中学学生，随学校组织的旅行团来美国参加夏令营活动[26]。

事后[编辑]

从跑道上的另一架飞机上拍摄到的失事飞机冒出的浓烟

美国国家运输安全委员会已派遣调查员前往调查[28]。涉事机场在事故发生后关闭，原定降落在旧金山国际机场的航班改為降落到旧金山湾区的其他机场（诺曼·峰田圣何塞国际机场、奥克兰国际机场）、萨克拉门托国际机场、洛杉矶国际机场和西雅图－塔科马国际机场，旧金山国际机场的两条跑道大约于旧金山当地时间2013年7月6日15时30分（UTC时间2013年7月6日22时30分）重新开放。[29]

韩媒东亚日报下属A频道主持人尹庆民在报道该新闻说道“是2名中国人而不是韩国人在事故中死亡，从我们的立场看，真是万幸。”引发争议，导致A频道为主持人不当言论做出道歉[30]。
